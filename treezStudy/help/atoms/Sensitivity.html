Sensitivity

Description

The atom Models is the parent for all model and sub model atoms. A model atom implements the interface org.treez.model.interfaces.Model.

Class name

org.treez.model.atom.Models

Construction

You can create a new Models atom:
  • from the context menu of an existing Root atom.
  • by calling the factory method of an existing Root atom in a text file: 
    Root root = new Root("root");

Context menu

The following context menu actions will add a new child atom to an existing Root atom. Follow the links to get more information about the added child atoms:
  • Add Models
  • Add Studies
  • Add Results
